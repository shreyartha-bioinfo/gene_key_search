# Gene Key Search Tool

## Overview
The Gene Key Search Tool is a web-based application designed to facilitate the search for scientific papers related to specific gene names and keywords. It interacts with the National Center for Biotechnology Information (NCBI) database to retrieve relevant research articles, providing curated results and citations for ease of reference.

## Features
- **User-Friendly Interface**: HTML text boxes for inputting gene names and keywords.
- **Curated Results**: Results are filtered and curated to present the most relevant papers.
- **Citations**: Provides proper citations for each retrieved reference, ensuring users can accurately credit sources in their work.

## How to Use
1. Navigate to the Gene Key Search Tool interface.
2. Enter the desired gene names into the designated text box.
3. Input relevant keywords that pertain to your research query.
4. Click the "Search" button to retrieve results.
5. Review the curated list of papers displayed along with their citations.

## Dependencies
- HTML
- JavaScript
- NCBI API

## Installation
To set up the Gene Key Search Tool locally:
1. Clone the repository: `git clone https://github.com/shreyartha-bioinfo/gene_key_search`
2. Open `index.html` in a web browser.
